# NDI-pyraview-matlab
NDI-matlab signal viewer

## Installation

This software requires [NDI-matlab](https://github.com/VH-Lab/NDI-matlab) and [pyraview](https://github.com/VH-Lab/Pyraview) to be installed first.

1.  **Install Prerequisites**:
    *   Follow the installation instructions for [NDI-matlab](https://github.com/VH-Lab/NDI-matlab).
    *   Follow the installation instructions for [pyraview](https://github.com/VH-Lab/Pyraview).

2.  **Install NDI-pyraview-matlab**:
    *   Clone this repository:
        ```bash
        git clone https://github.com/VH-Lab/NDI-pyraview-matlab.git
        ```
    *   Add the `src/ndi` directory of this repository to your MATLAB path.
